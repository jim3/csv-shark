## csvshark

[Go](https://pkg.go.dev/std) CLI tool for analyzing [Wireshark] CSV exports...


## Usage

```bash
./csvshark <FILENAME>.csv
```

Plans for command-line flags:
- `-filter`: Filter by column name and value

```bash
149.88.17.129
192.168.0.173
149.88.17.129
149.88.17.129
192.168.0.173
149.88.17.129
192.168.0.173
```
